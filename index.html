lattices is a package for handling formal concepts (Open Hub page)

This project uses:

  • graphviz for generating uml graphics
  • pandoc, gpp and LaTeX for the tutorials

If you are using this project in research activities including publication, you have to cite it using (BibTeX format).

  • authors: Karell Bertet, Christophe Demko, Jean-François Viaud, Clément Guérin
  • title: java-lattices: a Java library for lattices computation
  • year: 2014
  • how published: http://thegalactic.org
